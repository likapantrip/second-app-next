## アプリ作成目的
Next.jsの基本機能である「画面表示」と「ページ遷移」の理解・実装確認を目的とする。

## アプリ構成
```
app/
 ├ page.tsx          → /
 └ create/
     └ page.tsx      → /create
```

## 使用技術
- Next.js 16
- React
- Node.js 20
- App Router
